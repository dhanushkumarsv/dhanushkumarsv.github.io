# Dhanush Kumar S V — Chemical Engineering Portfolio

Personal portfolio of **Dhanush Kumar S V**, Process Engineer — live at
[dhanushkumarsv.github.io](https://dhanushkumarsv.github.io).

## 3D Process Plant Theme

A one-of-a-kind chemical-engineering portfolio built with HTML, CSS,
vanilla JavaScript and Three.js (no build step — GitHub Pages ready):

- **Interactive 3D process plant hero** — distillation columns, a shell &
  tube heat exchanger, a storage tank and pipework with glowing product
  flow, modelled in Three.js. Drag to rotate; it adapts to light/dark theme
- **3D flip cards** for technical expertise and **3D tilt** on cards
  throughout the site
- **Process-flow timeline** — internships displayed as instrument-circle nodes
- **Light theme by default** with a dark-mode toggle and saved preference
- **Downloads section** — resume PDF and vCard contact card
- Scroll progress bar, scroll-spy navigation, animated stats and GPA bars
- Fully responsive with reduced-motion support

## Structure

```
index.html      — single-page portfolio
style.css       — theme (light default + dark mode)
script.js       — Three.js plant scene, typed roles, tilt/flip, scroll effects
vendor/         — three.min.js (r128, vendored locally)
assets/         — downloadable resume PDF and vCard
images/         — profile photo
```
